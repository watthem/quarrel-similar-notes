# Screenshot Shot List

Capture these screenshots from the quarrel-test-vault in Obsidian.
Use a clean dark theme. Crop to the relevant UI area (no full desktop).

## Required Shots

### 1. `sidebar-overview.png` (Hero shot for README)
- Open a note with 3-5 similar results visible
- First card expanded, rest collapsed
- Keyword chips visible above results
- Filters panel collapsed
- **Crop**: Right sidebar panel only, ~400px wide

### 2. `keyword-chips.png`
- Same view as above but with 1-2 chips actively selected (highlighted)
- Results should visibly change from the filter
- **Crop**: Chips area + top 2 result cards

### 3. `card-expanded.png`
- One result card fully expanded showing:
  - Title + date
  - Similarity progress bar with percentage
  - Content snippet text
  - Tag pills (#tag1 #tag2)
  - "Open note" button
- **Crop**: Single expanded card, tight

### 4. `filters-panel.png`
- Filters panel expanded (click chevron)
- Both sliders visible: Max Results and Min Similarity
- **Crop**: Filters section only

### 5. `stale-banner.png`
- Modify a note so the stale banner appears
- Banner should show "X notes changed" with re-index link
- **Crop**: Bottom of sidebar showing banner + index status

### 6. `settings.png`
- Plugin settings tab open
- All 7 settings visible including Experimental section
- **Crop**: Settings pane, full height

### 7. `empty-state.png` (Optional)
- Panel open with no index built
- Shows "Index not built" message with instruction
- **Crop**: Sidebar panel

## Tips

- Use Obsidian's default dark theme for consistency
- Window width ~1200px so the sidebar is a natural size
- If using macOS, Cmd+Shift+4 then Space to capture a window
- Name files exactly as listed — the README references these paths
